nansat.node module¶

class nansat.node.Node(tag, value=None, **attributes)¶

Everything is a Node. The XML is maintained as (very efficient) Python objects until an XML representation is needed.

nodeList(tag)¶

Produce a list of subnodes with the same tag. Note: It only makes sense to do this for the immediate children of a node. If you went another level down, the results would be ambiguous, so the user must choose the node to iterate over.
